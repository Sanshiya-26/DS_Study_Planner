# DS Study Planner

A simple web-based study planner to generate a 100-day study plan for data science topics. Users can input their study preferences, and the application will suggest topics based on those inputs.

---

## How It Works

1. User inputs:
   - Career Goal
   - Study Hours per Day
   - Study Days per Week
   - Preferred Style (Article/Practice)

2. The app predicts a study topic based on the inputs and generates a daily study plan for 100 days.

3. Each day includes a motivational quote and a recommended study topic.

4. The user inputs are logged in a CSV file (`user_logs.csv`).
