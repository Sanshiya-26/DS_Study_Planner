# 📚 DS Study Planner

A Data Science study planner project for managing daily study schedules using Python and HTML via a web interface. The planner generates a 100-day study plan based on user preferences and offers recommended resources.

---

## 🌟 Features
- Generate a personalized 100-day study plan.
- Save user preferences and logs.
- Display motivational quotes for each day.
- Recommended study resources based on selected topics.
- Web interface for easy interaction.

---

## 🛠️ Prerequisites
- Python 3.x
- Flask
- Joblib
- Pandas
- Scikit-learn
